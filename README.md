# ez-ffmpeg
A shell script to make [ffmpeg](https://ffmpeg.org/) simpler to use.

> It can be used to connvert videos to .webp format (lossless)

### Usage:

```bash

git clone https://github.com/notarkhit/ez-ffmpeg.git
cd ez-ffmpeg/
./ez-ffmpeg.sh path/to/video

```
